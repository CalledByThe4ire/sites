# Function Components

Function components are the simplest way to write components in React. They don't hold state, they don't fire lifecycle events; they simply take props and render a React Element. In this course you'll learn all about function componentsâ€”writing them, composing them, using features like propTypes, context, and JSX spread attributes... all the things function component.
